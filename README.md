## 🚀 Getting Started

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/portfolio-bento.git
cd portfolio-bento
```

2. **Install dependencies**
```bash
npm install
# or
yarn install
# or
pnpm install
```

3. **Run the development server**
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

4. **Open your browser**
```bash
Navigate to http://localhost:3000 - Your portfolio awaits! 🎉
```
